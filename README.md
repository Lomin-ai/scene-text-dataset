# Lomin Scene Text Dataset

### 데이터셋 소개

Lomin Scene Text Dataset 데이터셋은 실생활에서 자주 볼 수 있는 다양한 환경의 글자들을 활용한 OCR 데이터셋입니다.

이 데이터셋은 2019년 과기정통부에서 주최한 그랜드 AI 챌린지에서 우승한 Lomin 팀이 후속과제의 지원을 받아 생성되었습니다.
